# ライフプラン・シミュレーター（FP Studio）

子育て・住宅・新NISA・老後まで含めた**生涯キャッシュフロー**をブラウザ上でシミュレーションするツールです。
左パネルのスライダーを動かすと、右側のグラフ（資産推移・キャッシュフロー・NISA枠）が**リアルタイム**に更新されます。

FP（ファイナンシャルプランナー）相談で使うことを想定し、世帯の収入・支出・教育費・住宅ローン・資産運用を
一画面で俯瞰できます。

## 使い方

```bash
npm install
npm run dev      # http://localhost:5173
```

ビルド:

```bash
npm run build    # dist/ に出力
npm run preview
```

入力内容はブラウザの localStorage に自動保存されます（`初期値に戻す`でリセット）。

## できること

- **基本キャッシュフロー** — 世帯（本人＋配偶者）の**額面年収**・昇給・年金・退職金から年間収支と資産残高を算出。所得税・住民税・社会保険料（介護保険は40歳〜）を自動控除して手取りに換算
- **子育て・教育費** — 「本人が何歳のときの子か」で出産年齢を設定。子どもごとに保育園・幼稚園（公立/私立/**インター（年額編集可）**）〜大学（国公立/私立文系/私立理系・一人暮らし）を選び、**中学受験・大学受験の塾**もON/OFFで学年別費用を自動計上
- **住宅** — 「賃貸で続ける」と「住宅を購入する（頭金・金利・返済期間つき元利均等ローン）」を切り替えて比較。物件価格は最大3億円まで
- **海外旅行・レジャー** — 毎年の旅行予算＋大型旅行の周期を設定
- **カスタム支出** — 車の買い替え・介護・習い事など、独自の費目を月額/年額/スポット（周期つき）で自由に追加
- **積立投資（NISA・iDeCo・特定口座）** — 「◯〜◯歳は月△万」と**期間ごとの積立プラン**を複数設定可能。**新NISA・iDeCoとも本人・配偶者それぞれ別枠**（NISAは各生涯1,800万＝夫婦3,600万、超過分は特定口座へ）。利回りは全口座共通。NISAは運用益非課税、特定口座は取り崩し時に約20.3%課税
- **iDeCo** — 本人・配偶者それぞれ設定可。**掛金は所得控除になり毎年節税**（税が下がり手取りが増える）、**65歳まで引き出せない**（それまでは取り崩し対象外）。受取時の税は退職所得控除等でほぼ非課税とみなす簡略化
- **現在の資産も本人・配偶者で分けて入力** — NISA・iDeCo・特定口座をそれぞれ。配偶者の既存NISAは配偶者の枠としてカウント（現金は世帯共通）
- **取り崩しの自動化・可視化** — 現金が足りない年は年齢に関係なく、特定口座→NISAの順に必要分だけ自動売却（**現金も投資も尽きて初めて資金ショート**）。取り崩しは専用の「取り崩し額」グラフ（手取り／譲渡税の内訳）・キャッシュフロー図・表の取崩列・資産推移の「取崩開始」ラインで、どの年にいくら取り崩したかを可視化
- **老後・年金** — 公的年金（老齢基礎年金＋老齢厚生年金の報酬比例部分）を**年収から自動概算**（手動入力に切替も可）。退職後は年金収入と資産の取り崩しで生活費をまかなう様子を可視化
- **資金ショート判定** — 現金が底をつく年を自動検知して警告

## グラフ

| グラフ | 内容 |
| --- | --- |
| 資産の推移 | 現金・預金＋特定口座＋NISA を積み上げた純資産の生涯推移。退職年に基準線 |
| 年間キャッシュフロー | 収入（給与・年金・退職金・取り崩し、上）と支出内訳（生活・住居・教育・旅行・その他、下）＋年間収支の折れ線。退職で給与が消え年金だけ残る様子が分かる |
| 投資資産（NISA・特定口座） | NISA と特定口座の評価額を積み上げ。非課税枠1,800万の基準線つき |
| 取り崩し額（投資の売却） | 現金不足を投資で補った額を年ごとに表示（手取り／譲渡税の内訳）。取り崩しが発生する場合のみ表示 |
| キャッシュフロー表 | 全年次の収入・支出・収支・現金・投資・純資産・ライフイベント一覧 |

## 技術

- React 19 + TypeScript + Vite
- Tailwind CSS v4（warm-paper のエディトリアル配色）
- Recharts（チャート）
- Zustand（状態管理＋localStorage永続化）

シミュレーションのコアは `src/engine/`、計算前提は [`CLAUDE.md`](./CLAUDE.md) を参照してください。

## 注意

金額はすべて **万円**。年収は **額面** で入力し、所得税・住民税・社会保険料を自動控除して可処分所得に
換算します（概算）。年金は年収から自動概算（基礎年金満額＋厚生年金 概算式、上限あり）、退職金は受取額で入力します。教育費は文部科学省「子供の学習費調査」等を
参考にした概算で、実際の制度・地域・進路により変動します。税・社会保険料の計算も配偶者控除等を省いた
簡略版です。計画づくりの**目安**としてご利用ください。
